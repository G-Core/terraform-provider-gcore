Known Issues

Provider-wide import drift on project_id / region_id
Optional path parameters (project_id, region_id) get populated during import from the import ID string, but the user's HCL config typically doesn't specify them. Since both fields have RequiresReplace, Terraform plans a recreate. This affects all cloud resources in the provider, not specific to this PR.

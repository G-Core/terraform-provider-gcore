JSON Merge Patch Behavior

The provider uses JSON Merge Patch (RFC 7386) for tag updates:

  • Adding a tag: {"tags": {"new_key": "value"}}
  • Modifying a tag: {"tags": {"existing_key": "new_value"}}
  • Deleting a tag: {"tags": {"key_to_delete": null}}

This is the fix for Bug 2 - tags can now be properly deleted by setting them to null in the PATCH request.

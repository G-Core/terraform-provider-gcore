Known Issues / Notes

IssueSeverityDescription
POST returns minimal data LOW The DNS zone POST API returns only {id, warnings}. Computed fields (contact, serial, status, etc.) are null in state until the first Read refresh. This causes "Objects changed outside Terraform" on first plan, but no actual resource modifications are planned. Cosmetic only.
meta drift on import FIXED Server injects keys like dnssec_status, targeting, max_hosts into meta. Import now clears meta to avoid drift. User's config value takes precedence on next apply.

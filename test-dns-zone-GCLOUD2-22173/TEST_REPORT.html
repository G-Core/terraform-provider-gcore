Fix Applied

File: internal/services/dns_zone/resource.go, line 200 (Read function)

- err = apijson.Unmarshal(bytes, &data)
+ err = apijson.UnmarshalComputed(bytes, &data)

Why this works: UnmarshalComputed uses IfUnset behavior for computed_optional fields — it only populates them from the API response when they are null or unknown. If the user explicitly set a value (e.g., serial = 2024010100), it is preserved in state. If the user didn't set them, they get populated on first create (when unknown) and then preserved on subsequent reads.
